(2.41)  BlackGold Oil Company needs to ship oil each day through its pipelines in Hillbilly, MO to Beverly Hills, CA through any or all of its monitoring stations.  The network of its stations and pipelines is given below.  Each arc in the network represents a pipeline with a corresponding maximum capaity (in thousands of gallons), which is due to the different pipeline diameters.  Formulate and solve a linear program to determine the maximum amount of oil that can be shipped through this network from Hillbilly to Beverly Hills.

\begin{center}
\includegraphics[width=.6\textwidth]{BlackGold}
\end{center}

\begin{enumerate}
	\item Is there an intuitive way to find the optimal objective value for this problem?
	\newpage
	\item Can you describe a minimization problem that has the same objective value as this max flow problem?  (This is the ``dual'' problem.)
	\newline \vspace{2in}
	\item Does this problem specify supply and demand?  If not, how can we write our balance of flow constraints for the start and end nodes?  (Note there are multiple ways to do this)
	\newpage
\item Formulate the LP associated with this max flow model \newpage
\item Generalize your LP model from part 5 to abstract (set) notation \newpage


\end{enumerate}
